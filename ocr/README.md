> Get the text in the screenshot to the clipboard


## requirement

First run this command on a terminal:

1. brew install pngpaste
2. brew install node (v16 test ok)


## dependency
百度OCR，[相关API](https://cloud.baidu.com/doc/OCR/s/zk3h7xz52)
