> Displays system information about your Mac.
Contains UserName、Model、HostName、Primary Ethernet Address、IPv4 Address、MacOS、CPU Type、Physical Memory、Storage、Locale、Display Resolution、Display System Uptime、Serial Number.

1. type `about` to trigger
2. `⏎` to copy info
3. large type support
