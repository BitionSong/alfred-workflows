> 快速切换选择代理Rules


## Usage

type `w2` to trigger


- ⌥ ⏎ 访问WEB管理页面
- ⏎ 切换rule选择状态
- ⌘ C拷贝当前Rule具体配置项
- ⇧或⌘ Y或⌘ L进行Rule预览
- 输入关键词筛选下拉选项
- 部分设定【规则多选[x]，版本更新检查】


## install

npm install -g whistle
